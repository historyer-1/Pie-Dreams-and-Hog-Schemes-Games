# Pie Dreams and Hog Schemes
## 内容介绍

这是一个**角色攻击随机生成怪物**的小游戏，玩家身边围绕三颗法球，法球攻击敌人会将敌人击杀并获得分数。  
敌人如果触碰到玩家则游戏结束，分数会被累加直至游戏结束。  

## 代码介绍

本项目使用了`easyx`和`SFML`库，其中easyx库用作项目的主要图形绘制，而sfml则用于输出音乐和音效。  
代码运用**智能指针**代替裸指针以保证内存安全，使用面向对象思想封装了动画类、玩家类、敌人类等。  
基于**享元模式**，又抽象出动画资源类作为动画类的基类来实现动画资源复用和多次的I/O操作带来性能损耗。
